# Dotfiles

Repositorio de configuración del entorno AwesomeWM y otros programas

### Programas

- **WM:** AwesomeWM
- **Notificaciones:** Naughty
- **Terminal:** Kitty
- **Web:** Vivaldi
- **Música:** MPD
- **Música (CLI):** Ncmpcpp & mpc
